Off-Campus Login

Use of this page and all subscription databases is limited to currently registered Edmonds College students and currently employed Edmonds College staff/faculty.

If you experience login difficulties, please read this page and try again.

For assistance, please use our chat service to connect with a librarian.

Students, please login using your Triton ID (example: s.student1234) and a permanent, self-selected password. Do NOT use your ctcLink ID/Canvas credentials. If you have not activated your Triton ID, follow these instructions before using the library databases.

Faculty and staff, please login using your campus network credentials.

That Triton ID username and/or password was incorrect. Please note that Triton ID passwords auto-expire after 120 days. If you are unable to login, please try resetting your password. Need help? Contact a librarian for assistance.
